Gravity : 중력을 적용시키는 모드입니다
Rotate : 공이 각속도를 가지고 회전하는 모드입니다.
BlackHole : 자체적인 중력을 가진 블랙홀을 출현시킵니다. 
- 모든 공이 블랙홀쪽으로 힘을 받습니다. 
- 방향키를 가지고 블랙홀의 위치를 조절할 수 있습니다.
- BlackHole Power를 조정해 블랙홀이 빨아들이는 힘을 조절할 수 있습니다.
- Eating 모드 : 블랙홀이 공을 먹어치우기 시작합니다. 먹은 질량에 비례해 블랙홀이 커지고 점수를 획득합니다. 
- AutoSpawn : 블랙홀이 공을 먹으면 다시 스폰합니다. 
- Restart : 블랙홀을 초기 크기로 되돌립니다.
